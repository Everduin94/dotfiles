# Raycast

Raycast is installed by `install/Brewfile.macos-dotfiles`.

This package includes an encrypted Raycast export for `everduin94` at `~/.config/raycast/config.rayconfig` after stow.

If that is not you, replace it with your own exported `.rayconfig` before running `./install/raycast-import.sh`.

Use `./install/raycast-save.sh` to write a fresh Raycast export back to the real file behind `~/.config/raycast/config.rayconfig`.
